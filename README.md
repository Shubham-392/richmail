# Setu e-Mail
## _Only Mail APP with regional languages support_



![Build Status](https://img.shields.io/badge/github-repo-blue?logo=github)

Setu is a mail receving MTA accepts mail from outbound servers like gmail, outlook, yahoo,etc.



## Features

- Multiple languages translations for inbox emails.
- languages support for now- `Hindi` `English` 
- Receive attachments like `image` `video` `pdf's` upto 25MB 
- Calender support
